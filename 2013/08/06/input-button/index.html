在input text里回车会执行button的onclick

Aug 6 2013

form、input(text)、button， 浏览器为了方便，让他们有了千丝万缕的关系。。。。

测试在input(text)里回车，是否会执行button的onclick？

一. 无form, button(没有设置type)*

<input type="text" name="text" />
<button onclick="alert(navigator.userAgent)">click me</button>

在input(text)回车：

  • chrome 没有弹窗
  • ie7 没有弹窗
  • ie8 ie9 弹出alert

二. 无form, button(type=button)

<input type="text" name="text" />
<button onclick="alert(navigator.userAgent)" type="button">click me</button>

在input(text)回车：

  • chrome 没有弹窗
  • ie7 ie8 ie9没有弹窗

三. 有form, button(没有设置type)

<form>
<input type="text" name="text" />
<button onclick="alert(navigator.userAgent)">click me</button> 
</form>

在input(text)回车：

  • chrome 弹出alert
  • ie7 没有弹窗
  • ie8 没有弹窗
  • ie9 弹出alert

结论：

  1. button的type在ie7下默认为button，ie8+ 和chrome默认为submit
  2. 套着form，button(type=submit)，input回车的时候，ie下会执行button的onclick,chrome不会。注：真奇葩
  3. 不套form，button(type=submit)，input回车的时候，chrome、ie9下会执行button的onclick, ie7、ie8不会
  4. 使用button的时候尽量带上type, 不然会出现诡异问题~
